Verb English Case +Thing Example

✅ Akkusativ (always used with these prepositions)

  • durch, für, gegen, ohne, um

✅ Dativ (always used with these prepositions)

  • aus, bei, mit, nach, seit, von, zu

⚠️ Both (Akkusativ OR Dativ depending on movement)

  • an, auf, in, über, unter, vor, hinter, neben, zwischen

👉 Use Akkusativ → when there is movement (direction)

Question: Wohin? (where to?)

  • an → Ich gehe an den Tisch
  • auf → Ich lege das Buch auf den Tisch
  • in → Ich gehe in das Haus
  • über → Ich gehe über die Straße
  • unter → Die Katze läuft unter den Tisch
  • vor → Ich stelle mich vor das Haus
  • hinter → Er geht hinter das Auto
  • neben → Ich setze mich neben dich
  • zwischen → Ich stelle mich zwischen die Leute

👉 Use Dativ → when there is position (no movement)

Question: Wo? (where?)

  • an → Ich bin an dem Tisch
  • auf → Das Buch liegt auf dem Tisch
  • in → Ich bin im Haus
  • über → Die Lampe hängt über dem Tisch
  • unter → Die Katze ist unter dem Tisch
  • vor → Ich stehe vor dem Haus
  • hinter → Das Auto steht hinter dem Haus
  • neben → Ich sitze neben dir
  • zwischen → Ich stehe zwischen den Leuten

🧠 One-line rule

👉 Movement → Akkusativ (Wohin?)
👉 Position → Dativ (Wo?)
